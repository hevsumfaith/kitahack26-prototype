# **App Name**: StreamWise

## Core Features:

- Student Profile & History: Allows students to create accounts, manage personal information, and track their assessment history over time.
- Interactive Assessment Module: Engaging quizzes and tests designed to evaluate student interests, strengths, and aptitudes beyond academic grades.
- Comprehensive Stream Directory: Provides detailed information for each available Form 4 stream, including subjects, prerequisites, and potential career paths.
- AI-Powered Compatibility Engine: An intelligent AI tool that processes assessment results and student profiles to calculate and generate compatibility percentages for each Form 4 stream.
- Personalized Recommendation Dashboard: Presents a clear, personalized dashboard showing compatibility scores, key insights, and a top recommendation for the most suitable Form 4 stream.
- Secure Data Storage: A backend system to securely store student accounts, assessment responses, stream details, and recommendation results.

## Style Guidelines:

- Primary color: A serene blue-green (#29B0D6), representing intelligence, clarity, and encouraging guidance. Suitable for primary actions and key elements.
- Background color: A very light, almost off-white with a hint of the primary hue (#F0F5F5), ensuring a clean, uncluttered, and readable canvas for educational content.
- Accent color: A fresh greenish-cyan (#21C4AD), providing visual distinction and energy for highlighting important information or calls to action while maintaining harmony with the primary.
- Headlines font: 'PT Sans' (humanist sans-serif) for its modern yet approachable feel, adding a touch of personality to headings and titles.
- Body text font: 'Inter' (grotesque-style sans-serif) for exceptional readability and a neutral, objective aesthetic suitable for conveying detailed educational content and assessment questions.
- Use clean, line-art icons that are universally understandable and visually consistent with the app's focus on education and clear guidance.
- Adopt a responsive, modular layout that prioritizes clear information hierarchy and user focus, especially during assessments and result display.
- Subtle, smooth animations for transitions between assessment stages and for displaying recommendation results, enhancing the user experience without distractions.
